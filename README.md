# Site Atlas

一个面向个人高频站点管理的导航项目，目标不是“保存更多链接”，而是“更快找到、理解并维护常用站点”。

## 开发

```bash
pnpm dev
```

访问 [http://localhost:3000](http://localhost:3000) 查看效果。

## 项目知识入口

- 仓库级执行入口：`AGENTS.md`
- 知识治理入口：`docs/README.md`
- 当前核心目标：先建立 AI 可持续遵循的知识治理系统，再实现导航产品

## 当前状态

- 已完成：知识治理系统骨架与首批项目资产
- 待开始：基于治理资产推进首页 MVP 实现
