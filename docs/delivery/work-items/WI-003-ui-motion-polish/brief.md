## 背景

首页在加入动效和主题切换后，工作区出现了未整理完成的 UI 改动。其中主题切换虽然已有入口，但实现链路存在断裂，导致切换后页面没有稳定响应。

## 目标

- 保留现有首页视觉方向
- 修复主题切换的初始化、点击切换和本地持久化链路
- 确保深浅色模式切换在当前浏览器中可持续生效

## 范围

- `app/layout.tsx`
- `app/globals.css`
- `components/ThemeToggle.tsx`

## 非目标

- 不重做整套首页视觉稿
- 不改动你尚未确认的其他 UI 细节
