# WI-002 简述

## 标题

推进 Site Atlas 首页 MVP

## 背景

P0 知识治理基座已经完成，项目已具备稳定的权威资产、任务路由和回写协议。下一阶段应进入 P1：首页 MVP，将前期定义的产品、信息架构、页面规格、数据模型和工程边界转化为可运行的页面能力。

## 目标

- 实现首页导航主链路
- 建立仓库内类型化站点数据源
- 支持搜索、分类筛选、置顶展示与卡片跳转
- 保持实现与现有权威资产一致

## 边界

### 本工作单包含

- 首页 MVP 的实施方案
- 首页所需数据结构与样例数据落地
- 首页组件与交互实现
- 最小必要验证与资产回写

### 本工作单不包含

- 书签导入
- 后台管理页面
- 登录、数据库、多用户能力
- 主题系统与高级个性化配置

## 关键路径

- 先落数据与过滤逻辑，再落页面结构与交互
- 保持搜索、分类、置顶三套规则一致
- 以最小客户端范围承载交互，不放大复杂度
