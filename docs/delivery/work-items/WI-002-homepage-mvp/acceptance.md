# WI-002 验收标准

- 首页展示清晰的产品标题与用途说明
- 首页具备搜索输入与分类切换
- 搜索可覆盖名称、描述、标签
- 分类筛选与搜索条件可叠加生效
- 用户可在当前浏览器置顶/取消置顶站点
- 刷新页面后置顶状态仍保留
- 存在可见置顶时，置顶站点在当前结果列表内优先展示
- 页面不渲染独立置顶区块
- 站点卡片至少展示名称、描述、分类、标签
- 点击卡片可在新标签页打开目标站点
- 仅 `active` 状态条目进入默认展示
- 置顶状态不写入站点内容模型
- 置顶站点与普通站点使用一致的卡片布局
- 代码结构与 `PRD`、`IA`、`HOME`、`DATA_MODEL`、`ARCHITECTURE` 保持一致
- 若实现导致权威资产发生变化，已同步回写
