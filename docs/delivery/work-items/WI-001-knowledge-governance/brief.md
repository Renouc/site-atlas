# WI-001 简述

## 标题

建立 Site Atlas 的知识治理系统

## 背景

项目刚初始化，若直接开始写 PRD 或实现页面，后续极易出现需求、设计、数据和实现口径不一致的问题。用户明确要求优先建立一套“AI 能始终遵循”的最佳设计。

## 目标

- 建立知识治理的执行入口
- 定义权威资产与优先级
- 建立任务路由与回写协议
- 产出首批产品、设计、工程、运营资产

## 边界

### 本工作单包含

- 根级执行入口补强
- `_meta` 治理元数据
- 首批核心资产
- 当前工作单本身

### 本工作单不包含

- 首页 UI 实现
- 数据文件与组件实现
- 书签导入能力
