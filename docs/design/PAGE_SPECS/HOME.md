# 首页规格

## 1. 页面目标

让用户在一个页面中完成站点查找、筛选和跳转，形成比浏览器收藏夹更高效的访问体验。

## 2. 页面结构

### 2.1 页头

- 产品名称
- 一句话说明

### 2.2 搜索区

- 搜索输入框
- 可选辅助说明，例如搜索范围提示

### 2.3 分类区

- “全部”选项
- 固定分类列表

### 2.4 置顶区

- 展示 `pinned = true` 的站点
- 若无置顶站点，则整个区块可不展示

### 2.5 结果区

- 展示经过搜索与筛选后的卡片列表
- 卡片按置顶优先、排序字段、名称等规则稳定展示

### 2.6 空状态区

- 当无结果时展示
- 引导用户修改搜索词或切换分类

## 3. 页面行为

- 输入搜索词后即时过滤
- 点击分类后叠加分类条件
- 点击站点卡片以新标签页打开目标站点

## 4. 展示优先级

- 置顶优先
- 同级按显式排序字段
- 仍无排序字段时按名称稳定排序

## 5. 验收口径

- 首页在无数据异常情况下可独立完成主任务
- 搜索、分类、置顶、结果展示逻辑相互一致
- 页面结构与 `PRD`、`IA`、`DATA_MODEL` 一致
