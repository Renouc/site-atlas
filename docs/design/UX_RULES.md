# 交互规则

## 1. 搜索规则

- 搜索为首页主入口，位置应足够显著
- 搜索范围覆盖名称、描述、标签
- 搜索反馈应即时，无需单独提交按钮

## 2. 筛选规则

- 分类筛选与搜索条件叠加生效
- 默认分类为“全部”
- 切换分类不应清空搜索词

## 3. 置顶规则

- 置顶用于承载用户最高频访问站点
- 置顶站点同时也应保留在完整结果中，除非页面设计明确做去重策略
- 置顶区数量应控制，避免抢占主内容空间

## 4. 卡片规则

- 卡片主要任务是帮助用户快速判断“要不要点开”
- 描述应保持短句，优先说明用途
- 标签不宜过多，避免视觉噪声

## 5. 空状态规则

- 无搜索结果时，提示用户调整关键词或切换分类
- 不出现“页面空白”的无反馈状态

## 6. 响应式规则

- 移动端优先保证搜索和卡片阅读
- 分类切换在窄屏下仍应清晰可操作
- 不依赖 hover 才能获得关键信息
