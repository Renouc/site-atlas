# 产品愿景

## 一句话愿景

把“越来越难管理的浏览器收藏夹”升级为“可回找、可置顶、可持续维护的个人收藏导航”。

## 用户价值

- 更快回忆：通过描述、标签、分类而不是仅靠站点名回忆
- 更快访问：通过首页导航替代深层收藏夹路径
- 更贴合个人习惯：常用站点可本地置顶并在当前浏览器持久化
- 更易维护：新增、归类、清理有统一规则
- 更可沉淀：站点集合从零散链接升级为个人知识资产

## 核心体验主张

- 首屏即搜索
- 搜索与分类组合使用
- 常用站点通过本地偏好层快速进入，而不污染内容层
- 没有置顶时页面保持干净，不出现空占位
- 每张卡片都能解释“它是什么、为什么值得留”

## 产品定位

- 定位：单人自用的收藏导航控制台
- 形态：单仓库维护、前端优先的轻量产品
- 风格：工具型、克制、以效率优先

## 长期方向

### V1

- 首页导航可用
- 支持搜索、分类、标签与本地置顶
- 内容以仓库内数据文件维护

### V2

- 支持书签导入
- 支持最近新增与精选合集
- 评估偏好导出或备份能力

### V3

- 在仍保持简单的前提下，评估内容维护后台或自动化流程
